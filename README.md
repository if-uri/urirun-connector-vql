# urirun-connector-vql

Native `vql://` URI surface over the [VQL](../../oqlos/vql) package: screenshot → structured scene (regions/objects with geometry, dominant colours, window title). Wayland-proof visual grounding that reads pixels, not the window manager.

See `../urirun/docs/URI_NATIVE_CONNECTOR_CHECKLIST.md` and `../urirun/docs/RESOLVE_FIRST_CONTROL_PLANE.md`.
